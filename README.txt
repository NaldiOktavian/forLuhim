PROJECT: Farewell Luhim Compact

Cara pakai:
1. Masukkan foto ke folder assets/img/
2. Gunakan nama:
   - foto-1.jpg
   - foto-2.jpg
   - foto-3.jpg
   - foto-4.jpg
   - foto-5.jpg
3. Masukkan lagu ke folder assets/music/
4. Nama lagu harus: lagu-luhim.mp3
5. Buka index.html di browser.

Kalau mau tambah foto:
- Tambah tag <img> baru di bagian slider pada index.html.
